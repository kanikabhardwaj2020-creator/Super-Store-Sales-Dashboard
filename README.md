# 📊 Super Store Sales Dashboard – Power BI Project

## 📌 Project Overview
This Power BI dashboard project analyzes Super Store sales data to generate meaningful business insights through interactive visualizations and KPI tracking.

The dashboard helps understand sales performance, profit trends, customer behavior, and future sales forecasting.

---

## 🎯 Objectives
- Create an interactive and visually appealing dashboard
- Analyze sales and profit performance
- Track KPIs and business metrics
- Generate sales forecasts using historical data
- Provide actionable business insights

---

## 🚀 Features
- Interactive filters and slicers
- Sales & Profit KPI Cards
- Region-wise and Category-wise analysis
- Monthly Sales Trends
- Sales Forecasting
- Dynamic Visualizations

---

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Excel

---

## 📈 Insights Generated
- Top-performing regions and products
- Sales growth patterns
- Profitability analysis
- Customer purchasing behavior
- Future sales predictions
